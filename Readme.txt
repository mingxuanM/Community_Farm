This app is Android only.

This app is developed for Sutton Community Farm to help them register volunteers and record volunteers' visitation.
To emulate the app, make sure you have Android APK, Sndroid Studio and an Android emulator with system API level no higher than 32 on your computer, and you need ionic installed to run the commend lines: 
Direct your Commend Prompt to the Sutton_Community_Farm folder and enter ’ionic emulate android’, ionic will open the app in your Android emulator.
Or you can use Android Studio to open the folder: *\Sutton_Community_Farm\platforms\android as an existing Android Studio project to run it.
